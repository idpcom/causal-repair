# External corpus: QuixBugs subset

Converted from QuixBugs (Koppel et al., MIT license,
https://github.com/jkoppel/QuixBugs) into the causal-repair task
format by bench/import-quixbugs.py. Held-out = the full official test
set; visible = the first fast-failing case. Contamination note: this
corpus is public and likely in model training data, which biases
results AGAINST the harness (memorized baselines look better) - a
conservative external-validity check.
